enum EHandsRepresentationMode
strong

The hand representation mode.

HANDS_SEGMENTATION is the default one, doing brightness-based segmentation. SEETHRU is not intended for regular use – only for comparison. ALTERNATIVE_SEGMENTATION is an experimental mode using colors to segment the hands.
