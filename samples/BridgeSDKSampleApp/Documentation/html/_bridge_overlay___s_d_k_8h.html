Parameters
clientNameThe name of the app using the Bridge DLL, for instance "myAwesomeGame". Whitespaces will be removed, if any.

BRIDGE_OVERLAY_SDK_API ESetHandsSegmentationThresholdErrorCode SetHandsSegmentationThreshold ( EHandsRepresentationMode  eHandsRepresentationMode,
float  segmentationThreshold 
)

Sets the hand segmentation threshold for a given EHandsRepresentationMode mode.

For the default HANDS_SEGMENTATION mode, this represents a lightness threshold above which pixels are displayed. The value is discarded for SEETHRU mode, and represents a hue width value for ALTERNATIVE_SEGMENTATION.
