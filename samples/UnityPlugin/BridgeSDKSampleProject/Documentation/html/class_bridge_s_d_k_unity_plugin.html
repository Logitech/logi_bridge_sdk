float  segmentationThreshold

inline

Sets the hand segmentation threshold for a given BridgeEnums.EHandsRepresentationMode mode.

For the default HANDS_SEGMENTATION mode, this represents a lightness threshold above which pixels are displayed. The value is discarded for SEETHRU mode, and represents a hue width value for ALTERNATIVE_SEGMENTATION.
